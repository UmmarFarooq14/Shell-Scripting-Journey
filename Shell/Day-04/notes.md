Control Structures & Loops in Shell Scripting:
----------------------------------------------
Introduction:
------------
	  In Programming, control structures allows us make to take decisions and perform loops.

  1) if, else, elif
  2) for loops

syntax  for if -ellse:
----------------------

	if [           ]
	then 
	    echo "       "
	else
	    echo "       "
	fi
 

syntax for elif:
----------------

	if [              ]
	then 
	    echo "        "
	elif [            ]
	then 
	    echo "        "
	else
	    echo "        " 
	fi
